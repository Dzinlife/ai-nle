# react-skia-lite

ESM version of react-native-skia for web.

## Installation

```bash
pnpm add react-skia-lite
```

## Usage

```tsx
import { /* components */ } from 'react-skia-lite';
```

## Development

This package is part of the ai-nle monorepo.

```bash
# Build
pnpm build

# Watch mode
pnpm dev
```
